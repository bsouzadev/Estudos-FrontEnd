Negrito/Destaque

Para deixar algo em negrito pode usar a tag b, porém essa tag b não é semantica

Uma tag semantica para o negrito é a strong

Italico / Enfase

Para deixar um termo em Italico se usa a tag i porém, não é semantica

Uma tag semantica para italico ou Enfase é a tag EM

Texto marcado

Se pode marcar um texto usando a tag mark

Texto grande e pequeno

Para texto grande se usa a tag BIG, e para texto pequeno se usa a tag small

Texto deletado

Podemos marcar um texto excluido usando a tag del

Texto Inserido

Podemos deixar um texto sublinhado usando a tag ins

Texto sobrescrito

Se usa sup. Ex: x2 + 4

Texto subscrito

Se usa a tag sub. Ex: H2O
